# JSONBin API Fetch Example

This project demonstrates fetching data from the JSONBin API using JavaScript, with a modern and visually appealing UI.

## Features
- Fetches data from https://api.jsonbin.io/v3/b/6977df03ae596e708ff88cef
- No anonymous functions, inline JavaScript, or inline CSS
- Responsive, modern design

## Usage
Open `index.html` in your browser.

## Project Structure
- `index.html` – Main HTML file
- `style.css` – External CSS for styling
- `app.js` – External JavaScript for API fetching and DOM updates
